# ADOT-v1

### instaluj pipenv

```
pip install pipenv
```

### spust pipenv

```
pipenv shell
```

### instaluj veci do pipenv 

```
pipenv install requests
pipenv install openpyxl
```

### selectni python interpreter na pipenv file

##### ctrl + shift + P 
##### search: select python interpreter
##### si ho nastav na ten ktory ma meno podla folderu
